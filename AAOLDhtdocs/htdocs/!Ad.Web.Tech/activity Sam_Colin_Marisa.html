The goal of this exercise is to download data from a remote source using AJAX and then upload this page.

jQuery is already loaded on this page. You should use it.

The first group to finish this wins a very special prize!

  1. Write you solution in the script tags at the bottom of this file
  2. Open http://horizons-json-cors.s3-website-us-east-1.amazonaws.com/data.json to check out what our data looks like.
  3. Load this data on this page using $.ajax().
  4. Find out which city is the largest by population.
  5. Using $.ajax, display the name of the city in the h1 tag at the top, it has the id "out".
  6. Rewrite the array (just include the first two objects) with the arrow in the JSON file in a XML format.
  7. Lastly, answer the following questions on your webpage or in the comments of your code: What is AJAX? What does it mean to be asynchronous? List 2 pros and 2 cons of AJAX. How do you write an AJAX function?
